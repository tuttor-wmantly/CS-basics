# Singly Linked List

#### Objective

* You cannot use lists or dictionaries
* Create two classes: `Node` and `LinkedList`
* Create your data structures and the following methods that allow your linked list to:
	- insert
	- search (First iteratively then recursively)
	- delete
	- print backwards
